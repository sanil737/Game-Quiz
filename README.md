A full-stack quiz application for gamers featuring real-time leaderboards, difficulty-based progression, and a coin-based hint system.

## Features
- **7 Games:** Free Fire, Clash Royale, CoC, PUBG, Among Us, Chess, Roblox.
- **Difficulty Scaling:** Questions move from Very Easy to Hard automatically.
- **Hint System:** 50-50, Textual, and First-letter hints (costs coins).
- **Monetization:** Google AdSense integration and Coin store.
- **No Login:** Just enter your name and play.

## Tech Stack
- **Frontend:** Vanilla JS / HTML5 / CSS3 (Mobile First)
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Ads:** Google AdSense

## Installation
1. Clone the repo.
2. Run `npm install` in the root.
3. Add your `MONGO_URI` to a `.env` file.
4. Run `node backend/server.js`.
